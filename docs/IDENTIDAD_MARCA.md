# Musify — Identidad de Marca

## Nombre

**Musify**

El nombre combina "music" con el sufijo "-fy" (del inglés, hacer/transformar), en la línea de Spotify o Shopify. La "y" final actúa como elemento visual diferenciador y se resalta en rojo en el logotipo.

---

## Logotipo

Logotipo tipográfico. El nombre "Musify" en tipografía serif con la **"y" final en rojo** (`#e63420`). No hay símbolo o icono separado — el propio texto es la marca.

Variante favicon: emoji musical 🎵 sobre fondo transparente en SVG inline.

Lema visible en la cabecera: **"Vol. 07 · 2026"** — evoca una edición discográfica, reforzando el concepto editorial de la marca.

## Paleta de colores

| Nombre | Variable CSS | Valor hex | Uso |
|--------|-------------|-----------|-----|
| Papel | `--paper` | `#efe6d2` | Fondo principal — cálido, como papel de disco antiguo |
| Papel medio | `--paper-2` | `#e7dcc1` | Fondos secundarios, tarjetas |
| Papel oscuro | `--paper-3` | `#ddd0ad` | Bordes, separadores |
| Tinta | `--ink` | `#14110b` | Texto principal — negro cálido |
| Tinta suave | `--ink-soft` | `#3d3527` | Texto secundario |
| Tinta muda | `--ink-mute` | `#7c705a` | Textos de apoyo, metadatos |
| Rojo (acento principal) | `--hot` | `#e63420` | Botones CTA, "y" del logo, elementos de acción |
| Rojo oscuro | `--hot-deep` | `#b22414` | Estado hover del rojo |

La paleta imita la estética de las portadas de discos de vinilo de los años 60-70: papel envejecido, tinta oscura y un rojo intenso como acento.

## Tipografías

| Rol | Familia | Uso |
|-----|---------|-----|
| Display / Titulares | **Instrument Serif** (Google Fonts) | H1, H2, nombres de canciones, elementos editoriales. Se usa en cursiva para énfasis. |
| Monoespaciada | **JetBrains Mono** (Google Fonts) | Metadatos, numeraciones, etiquetas, elementos técnicos |
| Sans-serif (sistema) | ui-sans-serif / Segoe UI | Texto de UI funcional (botones, labels, párrafos de interfaz) |

Las tipografías de display se cargan desde Google Fonts. Las demás son tipografías del sistema, sin dependencias externas.

## Tono de voz

- **Directo.** Sin rodeos. "Compón" en vez de "Empieza tu proceso creativo".
- **Creativo.** El lenguaje evoca el mundo musical sin tecnicismos: "estudio", "pista", "colección", "Vol. 07".
- **Sin jerga tecnológica** en la UI. El usuario no necesita saber qué es un JWT o un endpoint.
- **Cercano pero no infantil.** Primera persona en los mensajes de error y éxito: "Tu canción se guardó", "Algo salió mal".

---

## Aplicación visual

El diseño sigue una estética **editorial musical** — como la maquetación de las notas de un álbum vinilo. Las páginas usan mucho espacio en blanco (papel), texto grande y jerárquico, y el rojo como único color de acción. No hay gradientes, sombras dramáticas ni ilustraciones. El estilo es intencionalmente contenido y tipográfico.
