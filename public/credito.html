N.º 08 · Reconocimientos

Hecho con herramientas libres.

Musify no existiría sin el trabajo de miles de personas que publican código, fuentes y datos en abierto. Esto es lo que lo hace funcionar.

N.º 01 · Tecnologías

La pila completa

  1. 01

    PHP 8.3 + SQLite

    Backend API REST. Sin frameworks. PHP nativo con PDO para SQLite. Licencia MIT compatible.

  2. 02

    Ollama + Llama 3.2 3B

    Motor de IA para generación de letras. Corre 100% en local. Modelo publicado bajo licencia Meta Llama 3 Community License.

  3. 03

    MusicBrainz API

    Base de datos musical abierta con millones de artistas. Usada para búsqueda y enriquecimiento de metadatos. CC0 / ODbL.

  4. 04

    Instrument Serif · JetBrains Mono

    Tipografías de Google Fonts. Instrument Serif (Google LLC, OFL), JetBrains Mono (JetBrains, OFL).

  5. 05

    Web Crypto API · Web Audio API

    Estándares del navegador para hash de contraseñas (PBKDF2-SHA256) y síntesis musical generativa. Sin dependencias.

  6. 06

    validator.js · DOMPurify

    Validación de emails y sanitización contra XSS. Estándares de la industria recomendados por OWASP. MIT / Apache 2.0.
